## 🛠️ 技术栈

本博客使用以下技术构建：

- **VitePress** - 快速、现代的静态网站生成器
- **Vue 3** - 渐进式 JavaScript 框架
- **Markdown** - 简洁的标记语言
- **GitHub Pages** - 免费的静态网站托管

## 🚀 快速开始

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build

# 预览生产版本
npm run docs:preview
```

### 写作指南

1. 在 `docs/posts/` 目录下创建新的 Markdown 文件
2. 使用 YAML frontmatter 设置文章元数据
3. 支持 Markdown 扩展语法（表格、任务列表等）
4. 支持代码高亮和数学公式
